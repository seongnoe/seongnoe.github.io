1. 生成密钥

请先进入到你的C:\Users\Administrator\.ssh 文件夹下用git bash进行操作。

首先是先生成两个SSH密钥，对应你的两个身份(假设为 A 和 B)。

假设你当初已经用ssh-keygen -t rsa -C “youremail@qq.com”生成了一对秘钥id_rsa和id_rsa.pub ， 保存在了~/.ssh文件夹内。

然后，你需要再用另一个邮箱生成一对 private 的秘钥ssh-keygen -t rsa -C “B_eamil@qq.com”。这时候要注意重命名，否则会覆盖上面的密钥文件。

  • 应该怎么重命名？

假设我们生成了一对新的秘钥private 和private.pub。然后分别将这两对密钥加入到对应的 GitHub 账户中，这个步骤就略过了。

参考链接：https://www.cnblogs.com/leonliuo/p/16295806.html#label6

注：怎么你的密钥存放路径在哪？

  • Windows： C:\Users\Administrator\.ssh

  • Linux：/root/.ssh
